# Livestock Auction Platform - Coming Soon Page

A sleek, responsive coming soon landing page for the Livestock Auction Platform, designed to engage potential users and provide information about the upcoming service.

## Features

- **Modern, Responsive Design**: Looks great on all devices
- **Engaging Visuals**: Animated background with gradient effects
- **Feature Highlights**: Showcases key platform features
- **How It Works**: Simple step-by-step guide for users
- **Email Subscription**: Collect user interest for launch notifications
- **Contact Information**: Easy access to support

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Responsive Web Design
- Google Fonts (Segoe UI, Tahoma, Geneva, Verdana, sans-serif)

## Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, Edge)
- Web server (for local development)

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Navigate to the project directory:
   ```bash
   cd Testo-Co
   ```
3. Open `templates/coming-soon.html` in your web browser

## Usage

Simply open the `coming-soon.html` file in any modern web browser to view the landing page. No additional setup is required for the frontend.

## Features in Detail

- **Hero Section**: Eye-catching header with call-to-action
- **Feature Cards**: Highlighting key platform benefits
- **How It Works**: Step-by-step guide for users
- **Contact Form**: For user inquiries and subscription
- **Responsive Footer**: With copyright and additional links

## Customization

To customize the coming soon page:

1. Update the color scheme by modifying the CSS variables in the `<style>` section
2. Replace the placeholder text in the HTML with your content
3. Update the contact information in the footer
4. Modify the countdown timer (if implemented)

## Browser Support

The page is tested and works on:
- Google Chrome (latest)
- Mozilla Firefox (latest)
- Safari (latest)
- Microsoft Edge (latest)
